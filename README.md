# JAFR Administra tu negocio - Prototipo Web (Frontend)

Este repositorio contiene la maquetación web y el diseño de interfaz (UI/UX) para **JAFR**, un sistema administrativo orientado a pequeños emprendimientos de distribución de panadería tienda a tienda. 

El objetivo principal de esta interfaz es modernizar procesos arcaicos basados en papel, ofreciendo una experiencia de usuario fluida para el control de inventarios, gestión de clientes y registro de ventas/compras.

> **Nota del Proyecto:** Este repositorio representa **únicamente el alcance del Frontend** (Prototipo navegable en HTML/CSS/JS). No cuenta con conexión a base de datos ni backend, ya que su propósito es plasmar el diseño visual y la estructura de componentes basados en los prototipos de Figma.

## 🚀 Módulos de la Interfaz

La maquetación incluye las pantallas y modales para las siguientes funciones del sistema:

* **Gestión de Inventario:** Vistas para el control de stock actualizado.
* **Catálogos Principales:** Interfaces para la gestión de **Productos**, **Clientes** y **Proveedores**.
* **Movimientos:** Pantallas de registro para **Ventas** y **Compras**.
* **Reportes:** Interfaz visual con filtros para la futura generación de reportes (ventas, compras y reabastecimiento).
* **Roles de Usuario:** Diseño adaptado para dos perfiles:
    * *Administrador:* Acceso total a creación, edición y reportes.
    * *Vendedor:* Acceso de solo lectura a catálogos y permisos de creación únicamente para ventas.

## 🛠️ Stack Tecnológico

* **HTML5:** Estructuración semántica.
* **CSS3 & Bootstrap:** Estilos locales para diseño responsivo y componentes (sin dependencias de CDN externos).
* **JavaScript:** Interactividad básica de la interfaz (modales, menús desplegables).
* **Iconify:** Sistema de iconografía local.
